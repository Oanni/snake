# Игра "Змейка" (Snake Game)

Классическая игра "Змейка", реализованная на Python с использованием Pygame и принципов ООП.

## Описание

Игрок управляет змейкой, которая движется по игровому полю и собирает яблоки. С каждым съеденным яблоком змейка увеличивается в длину. Цель игры - набрать как можно больше очков, не столкнувшись с собственным телом.

## Особенности

- ✅ Реализация на принципах ООП
- ✅ Змейка проходит сквозь границы экрана (появляется с противоположной стороны)
- ✅ При столкновении с собой хвост отбрасывается, остается только голова
- ✅ Яблоко не появляется на теле змейки
- ✅ Отображение рекордной длины в заголовке окна
- ✅ Выход из игры по клавише ESC

## Требования

- Python 3.x
- pygame >= 2.0.0

## Установка

1. Клонируйте репозиторий:
```bash
git clone <URL_РЕПОЗИТОРИЯ>
cd snake
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python snake_game.py
```

- ⬆️ ⬇️ ⬅️ ➡️ - Стрелки для управления направлением движения змейки
